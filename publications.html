    Performance Study of the IceCube Upgrade Camera System
    ICRC 2025 C. Rott, M. Park, M. Jansson, G. Iverson, S. Choi (for the IceCube Collaboration)
    arXiv:2507.18525 — Instrumentation and Methods for Astrophysics
    Abstract

    The IceCube Upgrade Camera System is a calibration framework for measuring the optical properties of Antarctic ice with cameras and LEDs deployed on D-Egg and mDOM modules. These images can be used to study both local refrozen ice and bulk ice between strings, while also helping constrain module positions. This work compares classical maximum-likelihood and neural-network-based image analysis methods using a dedicated simulation pipeline.
